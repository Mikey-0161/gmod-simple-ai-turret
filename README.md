# GMod AI Turret

A lightweight AI turret entity for Garry's Mod.

## Features

- Automatically detects nearby NPCs
- Smooth target tracking
- Line-of-sight checks
- Configurable range, fire rate, and damage
- Uses default Half-Life 2 models, effects, and sounds
- No external content required

## Installation

1. Download or clone this repository.
2. Place the folder inside:
   garrysmod/addons/
3. Launch Garry's Mod.

## Spawning

Spawn the turret from:

Weapons → AI Turret

## Customisation

You can easily edit values such as:

- Health
- Damage
- Fire rate
- Detection range
- Rotation speed
- Search range

inside the entity's Lua file.
